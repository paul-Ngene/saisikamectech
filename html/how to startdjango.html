Django Multi-Project Workflow

1. Go to Your Main Projects Folder

cd ~/projects
This folder contains:
  • One shared virtual environment
  • Multiple Django projects

2. Create Shared Virtual Environment

python -m venv venv

3. Activate Virtual Environment

Linux / macOS

source venv/bin/activate

Windows CMD

venv\Scripts\activate

Windows PowerShell

.\venv\Scripts\Activate.ps1
Activate venv before working on any project.

4. Install Django

pip install django

Verify Installation

python -m django --version

5. Create New Django Project

mkdir ecommerce
cd ecommerce

django-admin startproject config .

6. Run Development Server

python manage.py runserver

Open:

http://127.0.0.1:8000/

7. Create Apps

python manage.py startapp users
python manage.py startapp products
python manage.py startapp payments

8. Register Apps

Open:

config/settings.py

Add apps inside INSTALLED_APPS:

INSTALLED_APPS = [
    "users.apps.UsersConfig",
    "products.apps.ProductsConfig",
    "payments.apps.PaymentsConfig",
]

9. Create Models

Example:

from django.db import models

class Product(models.Model):
    name = models.CharField(max_length=100)
    price = models.DecimalField(max_digits=10, decimal_places=2)

    def __str__(self):
        return self.name

10. Database Workflow

Create Migrations

python manage.py makemigrations

Apply Migrations

python manage.py migrate

11. Admin Setup

Create Superuser

python manage.py createsuperuser

Follow prompts:

Username:
Email:
Password:
This account logs into the Django admin dashboard.

12. Register Models in Admin

Open:

products/admin.py

Add:

from django.contrib import admin
from .models import Product

admin.site.register(Product)
Register every model you want visible in the admin dashboard.

13. Access Django Admin

Start Server

python manage.py runserver

Open Admin

http://127.0.0.1:8000/admin

Login with your superuser account.

14. Common Admin Customization

Example:

from django.contrib import admin
from .models import Product

@admin.register(Product)
class ProductAdmin(admin.ModelAdmin):
    list_display = ("id", "name", "price")
    search_fields = ("name",)
Useful for search, filters, sorting, and cleaner admin tables.
